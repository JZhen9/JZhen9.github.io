台中市YouBike

註冊方式

    註冊前請準備

  • 台灣手機號碼

    • 需透過手機簡訊接收驗證碼(手機號碼即是會員資料)

  • 身分證或居留證號碼

    • 須本人的身分證(居留證)號碼，作為公共自行車傷害險投保使用

  • 悠遊卡/一卡通

    • 官網僅提供電子票證綁定,欲使用性用卡或電子支付,請用YouBike2.0 APP註冊

    註冊流程

  • 1.0 APP

    • 點擊APP側選單⌈會員登入/註冊⌋>點擊頁面下方⌈註冊會員⌋>閱讀會員條款>填寫會員資料>設定卡片>完成註冊

  • 2.0 APP

    • 點擊APP側選單⌈會員登入/註冊⌋>點擊頁面下方⌈註冊會員⌋>閱讀會員條款>填寫會員資料>設定付款資訊>完成註冊

  • Kiosk自動服務機

    • 點擊功能列表的⌈我要租車⌋>點擊⌈加入會員⌋>閱讀會員條款>填寫會員資料>設定卡片>完成註冊

設備介紹

    1.0

  • U形把手與車頭:舒適輕巧好操控
  • 變速:提供三段變速，依騎乘狀況輕鬆切換。
    第1檔：適用於起步或上坡路段，騎乘感較輕巧。
    第2檔：適用於平路，騎乘舒適。
    第3檔：適用於平路加速或下坡路段，踩踏力度較重，下坡不易踩空。
    建議於上坡路段前先變速至第1檔，騎乘時較為省力。 若於第3檔時停車，起步前可提前變速至第1檔，輕鬆踩踏起步。
  • 旋轉車鈴:結合車把手成型，操作更便利安全!
  • 前後燈:前燈： 前輪驅動發亮LED頭燈、讓您行得安全。 後燈： 一體式嵌入、中途停靠可維持不滅60~90秒。
  • 低跨點下管:低跨點設計，上下車容易，嬌小身形也能輕鬆駕馭。
  • 輪胎:雙層輪圈，堅固耐用。
  • 後泥除:配色明亮顯眼，引人注目更安全!包覆式防吸裙設計，穿裙裝也能放心騎乘!*後泥除嚴禁乘坐， 限乘一人，禁止雙載喔!
  • 座墊:抗菌、防水、耐磨之優質座墊。
  • 座管高度調整桿:直覺式操作，按壓即可調整高度，輕鬆省力。
  • 隨車鎖:車頭向右旋轉，對準上鎖片孔與隨車鎖孔，將插銷對準上下鎖孔插入到底，即可完成。
  • 2.0 停車柱:太陽能板供電，具夜間警示功能。

騎乘須知
騎乘前檢查:
1.確認前後煞車功能:雙手漸進式調整煞車力道 切勿緊急煞車，以免造成危險
2.檢查前後輪胎壓:請確實檢查胎壓狀況
3.檢查車燈:確認前 / 後車燈功能正常
4.調整座墊:調整至適宜高度
5.座墊反轉 請勿租用:代表此輛故障待修 若發現車輛故障 將座墊"調最低"後"反轉" 並致電客服人員 將派員前往維修

騎乘小提醒:
1.不逆向騎乘
2.不併排聊天
3.不邊滑手機
4.不要空踩飛輪:未將車輛借出，僅於原地空踩，沒有健身功能，亦有可能導致車體受損
5.不違規雙載:YouBike 設計為單人騎乘使用，雙載容易發生危險，並造成車身損壞
6.記得帶走隨身物品:記得帶走置物籃內隨身物品，如有物品遺失請至官網「失物招領」查詢

坐墊調整教學
1.0:騎乘前請轉緊 並扣緊座管束子>向後扳開>轉動旋轉鈕調整鬆緊>調整高度>扣緊板手>往下壓確認座墊牢固
2.0:向上按壓，調整高度>放開拉桿，聽見喀聲>往下壓確認座墊牢固

公共自行車保險
  1. 傷害險

    • 投保方式

      新會員: 新會員於註冊時，填寫身分證 ( 居留證 ) 號碼及出生年月日等實名資料，第1張票卡 ( 悠遊卡、一卡通或電子支付 ) 即自動加入公共自行車傷害險，騎乘者無需負擔保費，惟 第 2~5 張票卡 / 既有舊會員 仍須自行填寫資料登錄。日後如有變更、新增票卡時，應確實調整投保資料，以確保自身權益。

      既有會員: 既有舊會員仍須自行填寫資料登錄，登錄方式共有 3 種：YouBike 官方 APP、官網及場站自動服務機 Kiosk。日後如有變更、新增票卡時，應確實調整投保資料，以確保自身權益

      1. 官方 App
      (1) 下載官方 App 並登入會員。
      (2) 打開側選單點擊「公共自行車傷害險」，進入卡片列表頁面。
      (3) 點擊未投保卡片的「加入保險」，進入投保流程。
      (4) 閱讀網路投保須知並正確填寫投保資料，即完成投保。

      2. 官方網站
      (1) 於官方網站登入會員。
      (2) 在「會員專區」頁面中點擊左側功能列表的「設定公共自行車傷害險」，進入卡片列表頁面。
      (3) 點擊未投保卡片的「加入保險」，進入投保流程。
      (4) 閱讀網路投保須知並正確填寫投保資料，即完成投保。

      3. 場站 Kiosk 自動服務機 ( 僅於有提供 YouBike 1.0 服務之地區有 )
      (1) 於租賃站操作 Kiosk 自動服務機，登入會員。
      (2) 點擊功能列表的「會員保險平台」，進入卡片列表頁面。
      (3) 點擊未投保卡片的「加入保險」，進入投保流程。
      (4) 閱讀網路投保須知並正確填寫投保資料，即完成投保。

    • 注意事項

      每一帳號可申請登記五張卡片 ( 含掃碼綁定卡 )，五張卡片可登記同一人為被保險人，或各自登記五位不同被保險人 ( 惟一卡僅能有一位被保險人 )，票卡實際使用者與被保險人須為同一人，並確認加保登錄資料，以確保享有傷害險之保障，發生意外事故時，方符合理賠資格。 提醒您騎乘時務必遵守交通規則，若有違反交通規則等因素，將不予理賠，倘若理賠申請及相關保險疑義可洽各地區保險公司，將有專責人員為民眾提供諮詢服務。

    • 保障範圍

      在所述地區 ( 註1 ) 使用以投保卡片 ( 含掃碼綁定卡 ) 租借YouBike，於騎乘時發生意外事故，造成被保險人死亡、失能 ( 殘廢 ) 及住院時，可憑醫療單據等文件，向保險公司申請保險理賠，保險金額最高理賠 100 萬元 ( 未滿 15 足歲者，依法僅給付失能保險金 )，傷害住院日額最高1,000 元。

    • 第三人責任險
      • 注意事項

        提醒您騎乘時務必遵守交通規則，若有違反交通規則等因素，將不予理賠，倘若理賠申請及相關保險疑義可洽各地區保險公司，將有專責人員為民眾提供諮詢服務。

      • 保障範圍

        若使用者於所述地區使用其所屬的 YouBike 車輛 ( 註 1 )，在租賃騎乘期間發生事故而導致第三人受傷或死亡時，傷者可憑警察機關處理證明紀錄、醫療單據等文件向保險公司申請保險理賠，保險金額為每一個人體傷或死亡最高理賠新台幣 200 萬元。
